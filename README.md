Este repositorio contiene el código correspondiente a los ejercicios del **Trabajo Práctico N°6** de la materia **Programación 2**.  

## Estructura del proyecto
- **src/**: Incluye el código fuente organizado por ejercicios.
  - **ejercicio1/**: Gestión de inventario y productos.
  - **ejercicio2/**: Gestión de biblioteca y libros.
  - **ejercicio3/**: Gestión de universidad, cursos y profesores.

## Cómo ejecutar
1. Compilar los archivos Java dentro de cada carpeta de ejercicio.
2. Ejecutar la clase `Main` correspondiente al ejercicio que se desea probar.
